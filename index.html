Big Cartel Front-End Dev Candidate Test

1. Write a JavaScript method to serialize the following form.

It should take a CSS selector as it's only argument and return an object whose structure matches { fieldname: 'fieldvalue' }. Disabled and blank fields should be ignored. Unmatched selectors should return an empty object. As this should be a simple method, no need to worry about nested forms or input arrays.

2. Flatten the object

Write a function that flattens a nested object into URLs, so that: 

{
  'index': {
    'about': {
      'team': true,
      'company': ['Jim', 'Barry']
    }
  }
}

Is tranformed into: 

{
  'index/about/team': true,
  'index/about/company': ['Jim', 'Barry']
}
